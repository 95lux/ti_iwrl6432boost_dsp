\doxysubsection{Detailed Description}
Initialization, configuration, and control functions for mm\+Wave sensor. 

This file contains functions to initialize, open, configure, start, stop, and deinitialize the mm\+Wave sensor. It also includes memory pool setup and HWA initialization.

\begin{DoxyNote}{Note}
Functionality derived from motion\+\_\+and\+\_\+presence\+\_\+detection\+\_\+demo\+\_\+xwr\+L64xx-\/evm\+\_\+m4fss0-\/0\+\_\+freertos\+\_\+ti-\/arm-\/clang.
\end{DoxyNote}

\begin{DoxyItemize}
\item Provides functions for mm\+Wave sensor control.
\item Includes memory pool initialization for object detection processing.
\item Handles HWA (Hardware Accelerator) initialization.
\item Implements calibration restore and validation using stored magic word.
\end{DoxyItemize}
